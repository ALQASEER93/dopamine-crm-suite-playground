# AGENTS — ALQASEER-PWA

## Commands
- `npm ci`
- `npm run build`

## PWA requirements
- Offline support: cache shell + queue writes (at least visits)
- Service worker must not break auth flows
- Don’t store secrets in localStorage
- Arabic-first, dark-mode friendly UI; respect GPS permissions/accuracy messages
- No destructive changes to caching or visit sync queues
